In order to setup CMSIS Configurator tool inside SEGGER Embedded Studio (SES):
  1. Open "tools.xml" file by going to File->OpenStudioFolder->External Tools Configuration
  2. Paste content of "cmsis_tool.xml" in between "tools" symbol inside "tools.xml" file
  3. Restart SES
  4. Right click on "sdk_config.h" -> CMSIS Configuration Wizzard
